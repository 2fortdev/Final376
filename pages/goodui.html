Original Layout



Here we can see what the current browse for classes page looks like. When clicking the subjects, a drop down menu appears with all the subjects that are avaliable.

The second form is not a drop down menu but an input for the class number. The third form is for you to search for a class via keywords.

The only thing it has here, is what I've typed previously.



The issues with this format is just usability, the subject list doesn't load in at once meaning as you scroll it has to stop and load more subjects. The second form gives no feedback or information what-so ever, making you type in the class number rather than showing you a list to choose from, this could help if you can only remember what level a class was.

The third form frankley does not work at all, I tried searching various subjects, class names, and no results were found any time I used it. It just seems a bit cumbersome and too light on the info for new users to handle.

Also as a personal pet peeve, left alligned text bothers me. So we'll get that centered. It's also just sort of floating there in a big container with the text, looks like the options are lost in the snow. Putting a container around it will focus the user's eyes on the stuff they need.
